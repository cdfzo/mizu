## Custom Icon Associations

You can customize the icon associations in the *settings.json* file:

```json
"mizu.icons": {
  "javascript": ["/scripts", "*.js", "*.cjs"],
  "readme": "readme.md"
}
```

Use the following formats for the different file types:

- Folder names: `/[name]`
- File extensions: `*.[name]`
- File names: `[name]`
